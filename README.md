# Postman-коллекция для API пиццерии

## Как использовать
1. Скачайте файл `API пиццерия.postman_collection.json`
2. Импортируйте в Postman (Import → Upload Files)
3. Готово!

## Доступные запросы
- GET /pizzas - получить все пиццы
- POST /pizzas - добавить новую пиццу
- GET /pizzas/{id} - получить пиццу по ID
- PUT /pizzas/{id} - обновить пиццу
- DELETE /pizzas/{id} - удалить пиццу
